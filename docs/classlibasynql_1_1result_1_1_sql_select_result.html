fixTypes ( array  $columns)

Some data in the rows may not be in the types intended, e.g. ints may be in string form. Use this method to fix the types.

Example usage:

$result->fixTypes(["name" => MysqlSelectResult::TYPE_STRING, "id" => MysqlSelectResult::TYPE_INT, "banned" => MysqlSelectResult::TYPE_BOOL])
Parameters
array$columnsan array of column names to TYPE_* constants.
